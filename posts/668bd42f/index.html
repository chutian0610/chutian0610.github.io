A point in program where the state of execution is known by the VMсђѓ

HotSpotуџёт«ЅтЁеуѓ╣т«џС╣Ѕ: Safepoint Тў»уеІт║ЈТЅДУАїУ┐ЄуеІСИГуџёСИђСИфуѓ╣№╝їтюеУ┐ЎСИфуѓ╣№╝їТЅђТюЅуџёGCТа╣жЃйТў»ти▓уЪЦуџё№╝їТЅђТюЅуџётаєт»╣У▒АтєЁт«╣жЃйТў»СИђУЄ┤уџёсђѓС╗јтЁет▒ђУДњт║дТЮЦуюІ№╝їТЅђТюЅуџёу║┐уеІжЃйт┐ЁжА╗тюет«ЅтЁеуѓ╣жў╗тАъ№╝їуёХтљјGCТЅЇУЃйУ┐љУАїсђѓ№╝ѕСйюСИ║СИђуДЇуЅ╣Т«іТЃЁтєх№╝їУ┐љУАїJNIС╗БуаЂуџёу║┐уеІтЈ»С╗Цу╗Ду╗ГУ┐љУАї№╝їтЏаСИ║т«ЃС╗гтЈфСй┐ућетЈЦТЪёсђѓтюет«ЅтЁеуѓ╣ТюЪжЌ┤№╝їт«ЃС╗гт┐ЁжА╗жў╗тАъУђїСИЇТў»тіаУййтЈЦТЪёуџётєЁт«╣сђѓ№╝ЅС╗јт▒ђжЃеУДњт║дТЮЦуюІ№╝їт«ЅтЁеуѓ╣Тў»С╗БуаЂтЮЌСИГуџёСИђСИфуЅ╣тѕФуѓ╣№╝їТЅДУАїу║┐уеІтЈ»УЃйС╝џтюеУ»Цуѓ╣жў╗тАъGCсђѓтцДтцџТЋ░ТЅДУАїуѓ╣жЃйугдтљѕт«ЅтЁеуѓ╣уџёТЮАС╗ХсђѓТ»ЈСИфт«ЅтЁеуѓ╣жЃйТюЅт╝║СИЇтЈўжЄЈ№╝їУ┐ЎС║ЏСИЇтЈўжЄЈтюежЮът«ЅтЁеуѓ╣тЈ»УЃйС╝џУбФт┐йуЋЦсђѓ

Тђ╗УђїУеђС╣І№╝їSafepoint ТїЄтюе Java УЎџТІЪТю║СИГ№╝їуеІт║ЈТЅДУАїТЌХуџёСИђСИфуЅ╣Т«іуѓ╣сђѓтюе Safepoint тцё№╝їТЅђТюЅуџёу║┐уеІжЃйС╝џУбФТџѓтЂюСИІТЮЦ№╝їС╗ЦСЙ┐У┐ЏУАїJVM уџёСИђС║ЏуЅ╣т«џуџёТЊЇСйю1сђѓСЙІтдѓ:

  • т«џТЌХУ┐ЏтЁЦ SafePoint№╝џу╗ЈУ┐Є-XX:GuaranteedSafepointIntervalжЁЇуй«уџёТЌХжЌ┤№╝їжЃйС╝џУ«ЕТЅђТюЅу║┐уеІУ┐ЏтЁЦ Safepoint№╝їСИђТЌдТЅђТюЅу║┐уеІжЃйУ┐ЏтЁЦ№╝їуФІтѕ╗С╗ј Safepoint ТЂбтцЇсђѓУ┐ЎСИфт«џТЌХСИ╗УдЂТў»СИ║С║єСИђС║ЏТ▓Ат┐ЁУдЂуФІтѕ╗ Stop the world уџёС╗╗тіАТЅДУАї№╝їТјеУЇљУ«Йуй«-XX:GuaranteedSafepointInterval=0тЁ│жЌГУ┐ЎСИфт«џТЌХ
  • ућ▒С║ј jstack№╝їjmap тњї jstat уГЅтЉйС╗ц№╝їС╣Ът░▒Тў» Signal Dispatcher у║┐уеІУдЂтцёуљєуџётцДжЃетѕєтЉйС╗ц№╝їжЃйС╝џт»╝УЄ┤ Stop the world№╝џУ┐ЎуДЇтЉйС╗цжЃйжюђУдЂжЄЄжЏєтаєТаѕС┐АТЂ»№╝їТЅђС╗ЦжюђУдЂТЅђТюЅу║┐уеІУ┐ЏтЁЦ Safepoint т╣ХТџѓтЂюсђѓ
  • тЂЈтљЉжћЂтЈќТХѕ(У┐ЎСИфСИЇСИђт«џС╝џт╝ЋтЈЉТЋ┤СйЊуџё Stop the world№╝їтЈѓУђЃ JEP 312: Thread-Local Handshakes):Java У«цСИ║№╝їжћЂтцДжЃетѕєТЃЁтєхТў»Т▓АТюЅуФъС║Ѕуџё№╝ѕТЪљСИфтљїТГЦтЮЌтцДтцџТЋ░ТЃЁтєхжЃйСИЇС╝џтЄ║уј░тцџу║┐уеІтљїТЌХуФъС║ЅжћЂ№╝Ѕ№╝їТЅђС╗ЦтЈ»С╗ЦжђџУ┐ЄтЂЈтљЉТЮЦТЈљжФўТђДУЃйсђѓтЇ│тюеТЌауФъС║ЅТЌХ№╝їС╣ІтЅЇУјитЙЌжћЂуџёу║┐уеІтєЇТгАУјитЙЌжћЂТЌХ№╝їС╝џтѕцТќГТў»тљдтЂЈтљЉжћЂТїЄтљЉТѕЉ№╝їжѓБС╣ѕУ»Цу║┐уеІт░єСИЇућетєЇТгАУјитЙЌжћЂ№╝їуЏ┤ТјЦт░▒тЈ»С╗ЦУ┐ЏтЁЦтљїТГЦтЮЌсђѓСйєТў»жФўт╣ХтЈЉуџёТЃЁтєхСИІ№╝їтЂЈтљЉжћЂС╝џу╗ЈтИИтц▒ТЋѕ№╝їт»╝УЄ┤жюђУдЂтЈќТХѕтЂЈтљЉжћЂ№╝їтЈќТХѕтЂЈтљЉжћЂуџёТЌХтђЎ№╝їжюђУдЂ Stop the world№╝їтЏаСИ║УдЂУјитЈќТ»ЈСИфу║┐уеІСй┐ућежћЂуџёуіХТђЂС╗ЦтЈіУ┐љУАїуіХТђЂсђѓ
  • Java Instrument т»╝УЄ┤уџё Agent тіаУййС╗ЦтЈіу▒╗уџёжЄЇт«џС╣Ѕ№╝џућ▒С║јТХЅтЈітѕ░у▒╗жЄЇт«џС╣Ѕ№╝їжюђУдЂС┐«Тћ╣ТаѕСИітњїУ┐ЎСИфу▒╗уЏИтЁ│уџёС┐АТЂ»№╝їТЅђС╗ЦжюђУдЂ Stop the worldсђѓ
  • Java Code Cache уЏИтЁ│№╝џтйЊтЈЉућЪ JIT у╝ќУ»ЉС╝ўтїќТѕќУђЁтј╗С╝ўтїќ№╝їжюђУдЂ OSR ТѕќУђЁ Bailout ТѕќУђЁТИЁуљєС╗БуаЂу╝ЊтГўуџёТЌХтђЎ№╝їућ▒С║јжюђУдЂУ»╗тЈќу║┐уеІТЅДУАїуџёТќ╣Т│ЋС╗ЦтЈіТћ╣тЈўу║┐уеІТЅДУАїуџёТќ╣Т│Ћ№╝їТЅђС╗ЦжюђУдЂ Stop the world
  • GC№╝џУ┐ЎСИфућ▒С║јжюђУдЂТ»ЈСИфу║┐уеІуџёт»╣У▒АСй┐ућеС┐АТЂ»№╝їС╗ЦтЈітЏъТћХСИђС║Џт»╣У▒А№╝їжЄіТћЙТЪљС║ЏтаєтєЁтГўТѕќУђЁуЏ┤ТјЦтєЁтГў№╝їТЅђС╗ЦжюђУдЂ Stop the world
  • JFR уџёСИђС║ЏС║ІС╗Х№╝џтдѓТъют╝ђтљ»С║є JFR уџё OldObject жЄЄжЏє№╝їУ┐ЎСИфТў»т«џТЌХжЄЄжЏєСИђС║ЏтГўТ┤╗ТЌХжЌ┤Т»ћУЙЃС╣Ёуџёт»╣У▒А№╝їТЅђС╗ЦжюђУдЂ Stop the worldсђѓтљїТЌХ№╝їJFR тюе dump уџёТЌХтђЎ№╝їућ▒С║јТ»ЈСИфу║┐уеІжЃйТюЅСИђСИф JFR С║ІС╗Хуџё buffer№╝їжюђУдЂт░є buffer СИГуџёС║ІС╗ХжЄЄжЏєтЄ║ТЮЦ№╝їТЅђС╗ЦжюђУдЂ Stop the worldсђѓ

SafePoint тдѓСйЋт«ъуј░№╝Ъ

SafePoint тЈ»С╗ЦТЈњтЁЦтѕ░С╗БуаЂуџёТЪљС║ЏСйЇуй«№╝їТ»ЈСИфу║┐уеІУ┐љУАїтѕ░ SafePoint С╗БуаЂТЌХ№╝їСИ╗тіетј╗ТБђТЪЦТў»тљджюђУдЂУ┐ЏтЁЦ SafePoint№╝їУ┐ЎСИфСИ╗тіеТБђТЪЦуџёУ┐ЄуеІ№╝їУбФуД░СИ║ Pollingсђѓ

уљєУ«║СИі№╝їтЈ»С╗ЦтюеТ»ЈТЮА Java у╝ќУ»ЉтљјуџётГЌУіѓуаЂуџёУЙ╣уЋї№╝їжЃйТћЙСИђСИфТБђТЪЦ Safepoint уџёТю║тЎетЉйС╗цсђѓу║┐уеІТЅДУАїтѕ░У┐ЎжЄїуџёТЌХтђЎ№╝їС╝џТЅДУАї Polling У»бжЌ« JVM Тў»тљджюђУдЂУ┐ЏтЁЦ SafePoint№╝їУ┐ЎСИфУ»бжЌ«Тў»С╝џТюЅТђДУЃйТЇЪУђЌуџё№╝їТЅђС╗Ц JIT С╝џС╝ўтїќт░йжЄЈтЄЈт░Љ SafePointсђѓ

у╗ЈУ┐Є JIT у╝ќУ»ЉС╝ўтїќуџёС╗БуаЂ№╝їС╝џтюеТЅђТюЅТќ╣Т│ЋуџёУ┐ћтЏъС╣ІтЅЇ№╝їтњїТЅђТюЅТЌауЋїтЙфуј»тЏъУи│С╣ІтЅЇТћЙуй«СИђСИф SafePoint(жў▓ТГбтЈЉућЪ GC жюђУдЂ Stop the world ТЌХ№╝їУ»Цу║┐уеІСИђуЏ┤СИЇУЃйТџѓтЂю)сђѓ

СйєТў»т»╣С║јТўјуА«ТюЅуЋїтЙфуј»№╝їСИ║С║єтЄЈт░Љ SafePoint№╝їТў»СИЇС╝џтюетЏъУи│С╣ІтЅЇТћЙуй«СИђСИфSafePoint№╝їСЙІтдѓ:

тюеТюЅуЋїforтЙфуј»жЄїжЮбТў»СИЇС╝џТћЙуй« SafePoint уџё№╝їУ┐ЎС╣Ът»╝УЄ┤С║єСИђС║ЏТђДУЃйС╝ўтїќуџёжЌ«жбўсђѓ

Т│еТёЈ№╝їС╗Ёжњѕт»╣ int ТюЅуЋїтЙфуј»№╝їтдѓТъюi у▒╗тъІТў»long У┐ўТў»С╝џТЈњтЁЦ SafePointсђѓ

СИіжЮбуџёС╗БуаЂуљєУ«║Тў»С╝џтюеprocessTask()тЄйТЋ░У┐ћтЏътЅЇТБђТЪЦ SafePointсђѓСйєТў»т«ъжЎЁСИіС╗ЇтЈ»УЃйСИЇУ┐ЏУАї SafePoint ТБђТЪЦсђѓтЏаСИ║ JIT С╝ўтїќтЈ»УЃйт»╝УЄ┤ SafePoint УбФТЊджЎц№╝їСЙІтдѓТќ╣Т│ЋУбФтєЁУЂћТЌХ№╝їт░▒СИЇС╝џУ┐ЏУАї SafePoint ТБђТЪЦсђѓ

тЈ»С╗ЦудЂуће JIT -Djava.compiler=NONE№╝їуюІуюІТў»тљдућ▒С║ј JIT т»╝УЄ┤сђѓ

SafePoint ТхЂуеІ

тюеJVMСИГ№╝їтГўтюеСИђСИфтЇЋСЙІуџётјЪтДІу║┐уеІVM Thread3№╝їт«ЃС╝џС║ДућЪтѕЏт╗║ТЅђТюЅтЁХС╗ќJVMу║┐уеІ№╝їтљїТЌХС╣ЪС╝џУбФтЁХС╗ќу║┐уеІућеС║јТЅДУАїСИђС║ЏТѕљТюгУЙЃжФўуџёVM Operions1сђѓУ┐ЎС║ЏVM OperationsС╝џС┐ЮтГўтюеVMOperationQueueСИГ№╝їжђџУ┐ЄУЄфУй«У»бТќ╣Т│Ћloop()У┐ЏУАїтцёуљєсђѓ

тюеVM ThreadжђђтЄ║тњїУй«У»бтцёуљєVM OperationsуџёУ┐ЄуеІСИГ№╝їжюђУдЂSTWТЮЦС┐ЮУ»Ђу╗ЊТъюуџётЄєуА«ТђД№╝їт░▒У░ЃућеSafepointSynchronize::begin()У┐ЏтЁЦSafePointсђѓ

  • VMThread::run() -> SafepointSynchronize::begin()№╝їТёЈтЉ│уЮђJVMтЇ│т░єтЁ│жЌГ№╝їТГцТЌХС╝џУ┐ЏтЁЦSafePoint№╝їУ┐ЏУАїжђђтЄ║тЅЇуџёТБђТЪЦ№╝їт╣ХуГЅтЙЁnative threadТЅДУАїт«їТ»Ћ№╝їТюђу╗ѕтЁ│жЌГТЋ┤СИфJVMсђѓТГцТЌХт░▒Т▓АТюЅУ░ЃућеSafepointSynchronize::end()
  • VMThread::loop() -> SafepointSynchronize::begin()сђѓтцёуљєт«їVM Operationsтљј№╝їжђџУ┐ЄУ░ЃућеSafepointSynchronize::end()жђђтЄ║SafePointсђѓ

ТјЦСИІТЮЦ№╝їтѕєТъљСИІSafepointSynchronize::begin()тєЁуџёТЊЇСйюсђѓућ▒С║јJVMу║┐уеІтЈ»УЃйтцёС║јСИЇтљїуџёуіХТђЂ№╝їСИ║С║єУЃйтцЪУ«Ет«ЃС╗гУЃйтцЪУ┐ЏтЁЦSafePoint№╝їJVMУ«ЙУ«АС║єСИЇтљїуџёТю║тѕХсђѓ
