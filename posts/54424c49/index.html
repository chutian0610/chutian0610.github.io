shell и„ҡжң¬йҖүйЎ№еҸӮж•°и§ЈжһҗйҖҡеёёжңү 3 з§Қж–№жі•гҖӮ

Linux shellдёӯеёёи§Ғзҡ„и„ҡжң¬еҸҳйҮҸ:

  • $0: еҚіе‘Ҫд»Өжң¬иә«
  • $N: з¬¬NдёӘеҸӮж•°пјҢдҫӢеҰӮ $1, $2, $3
  • $#: еҸӮж•°зҡ„дёӘж•°пјҢдёҚеҢ…жӢ¬е‘Ҫд»Өжң¬иә«
  • $@: еҸӮж•°жң¬иә«зҡ„еҲ—иЎЁпјҢдёҚеҢ…жӢ¬е‘Ҫд»Өжң¬иә«
  • $*: $*е’Ң$@зӣёеҗҢпјҢдҪҶ"$*"е’Ң"$@"е№¶дёҚеҗҢгҖӮ"$*" е°ҶжүҖжңүзҡ„еҸӮж•°и§ЈйҮҠжҲҗдёҖдёӘеӯ—з¬ҰдёІпјҢиҖҢ"$@"жҳҜдёҖдёӘеҸӮж•°ж•°з»„гҖӮ

shift жҳҜ bash зҡ„еҶ…зҪ®е‘Ҫд»ӨпјҢз”ЁдәҺз§»еҠЁеҸӮж•°ж•°з»„зҡ„дҪҚзҪ®гҖӮ

getopts

Bash зҡ„ getopts еҶ…зҪ®еҮҪж•°еҸҜд»Ҙе·§еҰҷең°и§Јжһҗе‘Ҫд»ӨиЎҢйҖүйЎ№гҖӮ

еӨ„зҗҶйҖүйЎ№

жҲ‘д»¬еңЁ while еҫӘзҺҜдёӯдҪҝз”Ё getoptsгҖӮеҫӘзҺҜзҡ„жҜҸж¬Ўиҝӯд»ЈйғҪеҜ№дј йҖ’з»ҷи„ҡжң¬зҡ„дёҖдёӘйҖүйЎ№иө·дҪңз”ЁгҖӮеңЁжҜҸз§Қжғ…еҶөдёӢпјҢеҸҳйҮҸ OPTION йғҪи®ҫзҪ®дёәз”ұ getopts ж ҮиҜҶзҡ„йҖүйЎ№гҖӮ

йҡҸзқҖеҫӘзҺҜзҡ„жҜҸж¬Ўиҝӯд»ЈпјҢgetopts з§»еҠЁеҲ°дёӢдёҖдёӘйҖүйЎ№гҖӮеҪ“жІЎжңүжӣҙеӨҡйҖүйЎ№ж—¶пјҢgetopts иҝ”еӣһ false е№¶дё” while еҫӘзҺҜйҖҖеҮәгҖӮ

OPTION еҸҳйҮҸдёҺжҜҸдёӘ case иҜӯеҸҘеӯҗеҸҘдёӯзҡ„жЁЎејҸзӣёеҢ№й…ҚгҖӮеӣ дёәжҲ‘д»¬дҪҝз”Ёзҡ„жҳҜ case иҜӯеҸҘпјҢжүҖд»ҘеңЁе‘Ҫд»ӨиЎҢдёҠжҸҗдҫӣйҖүйЎ№зҡ„йЎәеәҸж— е…ізҙ§иҰҒгҖӮжҜҸдёӘйҖүйЎ№йғҪиў«ж”ҫе…Ҙ case иҜӯеҸҘдёӯпјҢе№¶и§ҰеҸ‘йҖӮеҪ“зҡ„еӯҗеҸҘгҖӮ

getopts е‘Ҫд»ӨеҗҺи·ҹйҖүйЎ№еӯ—з¬ҰдёІгҖӮеҸӘжңүжӯӨеҲ—иЎЁдёӯзҡ„еӯ—жҜҚжүҚиғҪз”ЁдҪңйҖүйЎ№гҖӮжүҖд»ҘеңЁиҝҷз§Қжғ…еҶөдёӢпјҢ-d е°Ҷж— ж•ҲгҖӮиҝҷе°Ҷиў« ?) еӯҗеҸҘжҚ•иҺ·пјҢеӣ дёә getopts дёәжңӘиҜҶеҲ«зҡ„йҖүйЎ№иҝ”еӣһдёҖдёӘй—®еҸ·вҖң?вҖқгҖӮеҰӮжһңеҸ‘з”ҹиҝҷз§Қжғ…еҶөпјҢжӯЈзЎ®зҡ„з”Ёжі•е°Ҷжү“еҚ°еҲ°з»Ҳз«ҜзӘ—еҸЈгҖӮ

еҸҜд»ҘзңӢеҲ°и§ҰеҸ‘дәҶ usage еӯҗеҸҘпјҢдҪҶжҲ‘д»¬д№ҹд»Һ shell дёӯеҫ—еҲ°дёҖжқЎй”ҷиҜҜж¶ҲжҒҜгҖӮеҰӮжһңд»ҺеҸҰдёҖдёӘеҝ…йЎ»и§Јжһҗй”ҷиҜҜж¶ҲжҒҜзҡ„и„ҡжң¬и°ғз”ЁиҜҘи„ҡжң¬пјҢйӮЈд№ҲеҰӮжһң shell д№ҹз”ҹжҲҗй”ҷиҜҜж¶ҲжҒҜпјҢиҝҷе°ҶеҸҳеҫ—жӣҙеҠ еӣ°йҡҫгҖӮ

е…ій—ӯ shell й”ҷиҜҜж¶ҲжҒҜйқһеёёе®№жҳ“гҖӮжҲ‘д»¬йңҖиҰҒеҒҡзҡ„е°ұжҳҜе°ҶеҶ’еҸ·:дҪңдёәйҖүйЎ№еӯ—з¬ҰдёІзҡ„з¬¬дёҖдёӘеӯ—з¬ҰгҖӮ

еҪ“жҲ‘д»¬иҝҗиЎҢе®ғе№¶дә§з”ҹй”ҷиҜҜж—¶пјҢжҲ‘д»¬дјҡж”¶еҲ°жҲ‘д»¬иҮӘе·ұзҡ„й”ҷиҜҜж¶ҲжҒҜпјҢиҖҢжІЎжңүд»»дҪ• shell ж¶ҲжҒҜгҖӮ

йҖүйЎ№еҸӮж•°

иҰҒе‘ҠиҜү getoptsдёҖдёӘйҖүйЎ№еҗҺйқўе°Ҷи·ҹдёҖдёӘеҸӮж•°пјҢиҜ·еңЁйҖүйЎ№еӯ—з¬ҰдёІдёӯзҙ§и·ҹеңЁйҖүйЎ№еӯ—жҜҚеҗҺйқўж”ҫзҪ®дёҖдёӘеҶ’еҸ·:гҖӮеҪ“ getopt еӨ„зҗҶеёҰжңүеҸӮж•°зҡ„йҖүйЎ№ж—¶пјҢиҜҘеҸӮж•°иў«ж”ҫзҪ®еңЁ OPTARG еҸҳйҮҸдёӯгҖӮеҰӮжһңдҪ жғіеңЁи„ҡжң¬зҡ„е…¶д»–ең°ж–№дҪҝз”ЁиҝҷдёӘеҖјпјҢдҪ йңҖиҰҒе°Ҷе®ғеӨҚеҲ¶еҲ°еҸҰдёҖдёӘеҸҳйҮҸгҖӮ

ж··еҗҲйҖүйЎ№е’Ңеёёи§„еҸӮж•°

еҪ“ while еҫӘзҺҜйҖҖеҮәе№¶дё”жүҖжңүйҖүйЎ№йғҪе·ІеӨ„зҗҶеҗҺпјҢжҲ‘д»¬е°Ҷе°қиҜ•и®ҝй—®еёёи§„еҸӮж•°гҖӮ

getopt

getoptжҳҜдёҖдёӘеӨ–йғЁе‘Ҫд»ӨпјҢйҖҡеёёLinuxеҸ‘иЎҢзүҲдјҡиҮӘеёҰпјҢдҪҝз”Ёж—¶е»әи®®дҪҝз”Ёtype getoptжЈҖжөӢдёӢжңүжІЎжңүе®үиЈ…гҖӮзӣёжҜ”getopt,getoptж”ҜжҢҒзҹӯйҖүйЎ№е’Ңй•ҝйҖүйЎ№пјҢиҝҳжңүеҸҜйҖүеҸӮж•°гҖӮ

getoptжңүдёҚеҗҢзҡ„зүҲжң¬пјҢеўһејәзүҲ(enhanced)зӣёжҜ”дј з»ҹзҡ„getopt(д№ҹз§°дёәе…је®№зүҲжң¬зҡ„getopt)пјҢжҸҗдҫӣдәҶеј•еҸ·дҝқжҠӨзҡ„иғҪеҠӣ.иҰҒйӘҢиҜҒе®үиЈ…зҡ„getoptжҳҜеўһејәзүҲзҡ„иҝҳжҳҜдј з»ҹзүҲзҡ„пјҢдҪҝз”Ёgetopt -TеҲӨж–ӯеҚіеҸҜгҖӮеҰӮжһңе®ғд»Җд№ҲйғҪдёҚиҫ“еҮәпјҢеҲҷжҳҜеўһејәзүҲпјҢжӯӨж—¶е®ғзҡ„йҖҖеҮәзҠ¶жҖҒз Ғдёә4гҖӮеҰӮжһңиҫ“еҮә"вҖ“"пјҢеҲҷжҳҜдј з»ҹзүҲзҡ„getoptпјҢжӯӨж—¶е®ғзҡ„йҖҖеҮәзҠ¶жҖҒз Ғдёә0гҖӮеҰӮжһңжғіеңЁи„ҡжң¬дёӯиҝӣиЎҢзүҲжң¬жЈҖжҹҘпјҢеҸҜд»ҘеҸӮиҖғеҰӮдёӢд»Јз Ғпјҡ
getopt -T &>/dev/null;[ $? -ne 4 ] && { echo "not enhanced version";exit 1; }

дёҖиҲ¬жқҘиҜҙпјҢзҹӯйҖүйЎ№жҳҜеҸӘдҪҝз”ЁдёҖдёӘ"-вҖңејҖеӨҙпјҢйҖүйЎ№йғЁеҲҶеҸӘдҪҝз”ЁдёҖдёӘеӯ—з¬ҰпјҢй•ҝйҖүйЎ№жҳҜдҪҝз”ЁдёӨдёӘзҹӯжЁӘзәҝ(еҚівҖқвҖ“")ејҖеӨҙзҡ„гҖӮ

  • зҹӯйҖүйЎ№еҸҜд»ҘйҖҡиҝҮдёҖдёӘзҹӯжЁӘзәҝ"-вҖңе°ҶеӨҡдёӘзҹӯйҖүйЎ№иҝһжҺҘеңЁдёҖиө·пјҢдҪҶеҰӮжһңиҝһеңЁдёҖиө·зҡ„зҹӯйҖүйЎ№жңүеҸӮж•°зҡ„иҜқпјҢеҲҷеҝ…йЎ»дҪңдёәдёІиҒ”зҡ„жңҖеҗҺдёҖдёӘеӯ—з¬ҰгҖӮдҫӢеҰӮвҖқ-avz"е…¶е®һдјҡиў«и§Јжһҗдёә"-a -v -z"пјҢtar -zcf a.tar.gzдёІиҒ”дәҶеӨҡдёӘзҹӯйҖүйЎ№пјҢдҪҶ"-f"йҖүйЎ№жңүеҸӮж•°a.tar.gzпјҢжүҖд»Ҙе®ғеҝ…йЎ»дҪңдёәдёІиҒ”йҖүйЎ№зҡ„жңҖеҗҺдёҖдёӘеӯ—з¬ҰгҖӮ
  • зҹӯйҖүйЎ№зҡ„еҸӮж•°еҸҜд»Ҙе’ҢйҖүйЎ№еҗҚз§°иҝһеңЁдёҖиө·пјҢд№ҹеҸҜд»ҘжҳҜз”Ёз©әзҷҪеҲҶйҡ”гҖӮдҫӢеҰӮ-n 3е’Ң-n3жҳҜзӯүд»·зҡ„пјҢж•°еҖј3йғҪжҳҜ"-n"йҖүйЎ№зҡ„еҸӮж•°еҖјгҖӮ
  • еҰӮжһңжҹҗдёӘзҹӯйҖүйЎ№зҡ„еҸӮж•°жҳҜеҸҜйҖүзҡ„пјҢйӮЈд№Ҳе®ғзҡ„еҸӮж•°еҝ…йЎ»зҙ§и·ҹеңЁйҖүйЎ№еҗҚеҗҺйқўпјҢдёҚиғҪдҪҝз”Ёз©әж јеҲҶејҖгҖӮ
  • й•ҝйҖүйЎ№еҸҜд»ҘдҪҝз”ЁзӯүеҸ·жҲ–з©әзҷҪиҝһжҺҘдёӨз§Қж–№ејҸжҸҗдҫӣйҖүйЎ№еҸӮж•°гҖӮдҫӢеҰӮвҖ“file=FILEжҲ–вҖ“file FILEгҖӮ
  • еҰӮжһңжҹҗдёӘй•ҝйҖүйЎ№зҡ„еҸӮж•°жҳҜеҸҜйҖүзҡ„пјҢйӮЈд№Ҳе®ғзҡ„еҸӮж•°еҝ…йЎ»дҪҝз”Ё"="иҝһжҺҘгҖӮ
  • й•ҝйҖүйЎ№дёҖиҲ¬еҸҜд»Ҙзј©еҶҷпјҢеҸӘиҰҒдёҚдә§з”ҹжӯ§д№үеҚіеҸҜгҖӮ

дҫӢеҰӮпјҢlsе‘Ҫд»ӨпјҢд»Ҙ"a"ејҖеӨҙзҡ„й•ҝйҖүйЎ№жңү3дёӘгҖӮ

еҰӮжһңжғіиҰҒжҢҮе®ҡвҖ“almost-allпјҢеҸҜд»Ҙзј©еҶҷдёәвҖ“almпјӣеҰӮжһңжғіиҰҒжҢҮе®ҡвҖ“authorпјҢеҸҜд»Ҙзј©еҶҷдёәвҖ“auгҖӮеҰӮжһңеҸӘзј©еҶҷдёә"вҖ“a"пјҢbashе°Ҷз»ҷеҮәй”ҷиҜҜжҸҗзӨәпјҢй•ҝйҖүйЎ№еҮәзҺ°жӯ§д№үгҖӮ

еҸҜйҖүеҸӮж•°

жңүдёҚеҗҢзұ»еһӢзҡ„е‘Ҫд»ӨиЎҢйҖүйЎ№пјҢиҝҷдәӣйҖүйЎ№еҸҜиғҪдёҚйңҖиҰҒеҸӮж•°пјҢд№ҹеҸҜиғҪеҸӮж•°жҳҜеҸҜйҖүзҡ„пјҢд№ҹеҸҜиғҪжҳҜејәеҲ¶иҰҒжұӮеҸӮж•°зҡ„гҖӮеҰӮжһңжҹҗдёӘйҖүйЎ№зҡ„еҸӮж•°жҳҜеҸҜйҖүзҡ„пјҢйӮЈд№Ҳе®ғзҡ„еҸӮж•°еҝ…йЎ»дёҚиғҪдҪҝз”Ёз©әж је°ҶеҸӮж•°е’ҢйҖүйЎ№еҲҶејҖгҖӮеҰӮжһңдҪҝз”Ёз©әж јеҲҶйҡ”пјҢеҲҷж— жі•еҲӨж–ӯе®ғзҡ„дёӢдёҖдёӘе…ғзҙ жҳҜиҜҘйҖүйЎ№зҡ„еҸӮж•°иҝҳжҳҜйқһйҖүйЎ№зұ»еһӢзҡ„еҸӮж•°гҖӮ
дҫӢеҰӮпјҢ-cе’ҢвҖ“configйҖүйЎ№зҡ„еҸӮж•°жҳҜеҸҜйҖүзҡ„пјҢиҰҒеҗ‘иҝҷдёӨдёӘйҖүйЎ№жҸҗдҫӣеҸӮж•°пјҢеҝ…йЎ»еҶҷжҲҗ-cFILEгҖҒвҖ“config=FILEпјҢеҰӮжһңеҶҷжҲҗ-c FILEгҖҒвҖ“config FILEпјҢйӮЈд№Ҳе‘Ҫд»ӨиЎҢе°Ҷж— жі•еҲӨж–ӯиҝҷдёӘFILEжҳҜжҸҗдҫӣз»ҷйҖүйЎ№зҡ„еҸӮж•°пјҢиҝҳжҳҜйқһйҖүйЎ№зұ»еһӢзҡ„еҸӮж•°гҖӮ

unixзҡ„е‘Ҫд»ӨиЎҢдёӯпјҢжҖ»жҳҜеҸҜд»ҘеңЁйқһйҖүйЎ№зұ»еһӢзҡ„еҸӮж•°д№ӢеүҚеҠ дёҠ"вҖ“"пјҢиЎЁзӨәйҖүйЎ№е’ҢйҖүйЎ№еҸӮж•°еҲ°жӯӨдёәжӯўпјҢеҗҺйқўзҡ„йғҪжҳҜйқһйҖүйЎ№зұ»еһӢзҡ„еҸӮж•°гҖӮдҫӢеҰӮпјҡ
